



# Double_wishbone_concept_model

This suspension assembly was made using NX 2512 software and serves as a rough concept on how the double wishbone independent suspension layout functions.

<img width="1655" height="875" alt="Suspension_Assembly" src="https://github.com/user-attachments/assets/4bdc81ca-0186-49dc-bb94-c84f9836d885" />

The model was built by visual scale therefore the dimensions will not correlate to a precise scale. This model is merely a concept built to visualise the concepts of suspension travel, camber angle and various other mechanical properties. The ratios and relations are slightly exaggerated to visualise the different physical and mechanical concepts better. This can be seen in the video of the simulated movement of the suspension model.

https://github.com/user-attachments/assets/e720a7e0-6d6c-4f4d-9288-c52cc5f37098

The next iteration of the project would be made as a more realistic model for true independent suspension to customise and add it to other more complex integrated projects.
